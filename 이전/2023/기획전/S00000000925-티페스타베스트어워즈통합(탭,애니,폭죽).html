  • [선착순 더블쿠폰 유의사항]
  • '선착순 쿠폰 다운받기' 클릭 시 지급되며, 마이쇼핑>쿠폰에서 확인 가능합니다.
  • 선착순 더블쿠폰은 매일 오전 10시부터 선착순 10,000명까지 다운로드 가능합니다.
  • 선착순 더블쿠폰은 마케팅 수신동의 후 발급 가능합니다.
  • ID당 5,000원, 2,000원 각 1매씩 총 2매 발급 가능합니다.
  • 5,000원 더블쿠폰은 5만원 이상 [TV쇼핑] 상품 구매 시, 2,000원 더블쿠폰은 5만원 이상 전상품 구매 시 사용 가능합니다.
  • 쿠폰 적용상품은 일자별 상이할 수 있습니다. (마이페이지>쿠폰>'적용가능 상품 보기'에서 확인가능)
  • 더블쿠폰은 다른 쿠폰과 중복하여 사용하실 수 있습니다.
  • 쿠폰은 장바구니 단위로 적용되지 않으며, 개별 상품 단위로 적용 가능합니다.
  • 발급받은 선착순 쿠폰은 발급일시로부터 1일간 사용 가능합니다.
  • 해당 쿠폰은 쇼핑엔티 모바일 APP에서만 사용 가능합니다.
  • 가격비교사이트, 제휴몰 링크를 통해서 접속한 경우 더블쿠폰 사용이 불가합니다.
  • 발급받은 쿠폰은 주문 시 더블쿠폰을 선택해야 할인 적용 가능합니다.
  • 본 이벤트는 당사 사정에 의해 변경 또는 조기 종료될 수 있습니다.
